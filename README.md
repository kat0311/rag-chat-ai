# RAG Chat Application

A Flask + React chat application with RAG (Retrieval-Augmented Generation) capabilities.

## Features
- Chat interface with OpenAI integration
- Document upload and processing (PDF)
- RAG pipeline for intelligent retrieval
- Vector search with ChromaDB

## Tech Stack
- Backend: Flask, LangChain, ChromaDB
- Frontend: React
- AI: OpenAI GPT
